# Samsara Product

Samsara is a Personal OS for daily execution and long-term memory. It is designed for one owner and optimized for durability, clarity, and extensibility.

## Positioning

Samsara is not a personal homepage. The public profile is secondary. The core product is a private workspace used every day.

## Core Workflows

- Start from Dashboard
- Capture tasks in Todo
- Write structured Markdown in Documents
- Store and attach files through Files
- Record life, learning, and reflection in Diary
- Search across durable personal data
- Use AI Workbench through one centralized AI layer
- Configure owner settings from Settings

## Design Principles

- Private-first personal data
- Modular feature folders
- Shared infrastructure under `src/lib`
- Provider boundaries for storage and AI
- Database-backed records before advanced automation
- Small phases with explicit verification

## v1.0 Scope

v1.0 includes the full private workspace foundation. It intentionally does not include RAG, vector search, multi-user collaboration, cloud storage providers, or production deployment automation.
