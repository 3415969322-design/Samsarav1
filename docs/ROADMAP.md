- Public profile
- Password-protected Personal OS
- Dashboard
- Todo
- Documents
- Tags
- Search
- Files
- Diary
- AI Workbench

- Real AI context providers for Diary, Todo, Documents, and Files
- RAG and vector search after core data is stable
- Cloud storage providers for R2, S3, and Supabase Storage
- Advanced file organization assistant
- Daily/weekly review workflows
- Timeline and knowledge graph views
- Optional deployment automation
